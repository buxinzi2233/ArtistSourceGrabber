# Artist Source Grabber V2

V2 主要更新：

- 全面重做现代简约 WebUI；
- 图片按真实宽高比展示；
- 新增原图预览与逐图打标详情；
- X / Pixiv 专用登录 Profile 持久化和浏览器进程清理；
- 多来源统一目录、文件名前缀和 SHA-256 去重；
- LLM JSON 解析、预设提示词和 ONNX 标签合并策略修复；
- 新增完整依赖一键安装脚本。

## 安装

1. 下载并解压 `ArtistSourceGrabber-V2.zip`。
2. 先运行 `先运行这个.bat`。
3. 安装完成后运行 `start.bat`。
4. 浏览器打开本机 WebUI 后再配置来源、登录和打标器。
